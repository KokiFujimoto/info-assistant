# 開発履歴 (CHANGELOG)

## [Unreleased] - 2025-11-27

### ✨ 新機能 (Features)
- **AI分析パイプラインの実装**
  - Gemini API (gemini-2.0-flash-001) を使用した記事分析機能
  - 重要度スコアリング (0-100)
  - エンティティ抽出 (人名, 組織, 技術, イベント)
  - 感情分析 (Positive/Neutral/Negative)
  - 自動タグ付け
  - ベクトル埋め込み (3072次元)

- **ダッシュボードの強化**
  - AI分析結果の視覚化（重要度バッジ、感情アイコン）
  - 記事要約の表示
  - タグとエンティティの表示
  - **タグ/エンティティによるフィルタリング機能**
  - **Obsidian風ナレッジグラフの実装**
    - ダークモード＆サイバーなデザイン
    - エンティティ（人、組織、技術）のノード化と結合
    - 物理演算コントロールとフィルタリングパネル
    - ノードラベルの表示と振動防止
    - **全画面グラフページの追加 (`/graph`)**

- **既読管理機能**
  - 記事の既読/未読ステータス管理
  - 未読/既読/すべてのフィルタリングタブ
  - 既読時のフェードアウトアニメーション

- **UI/UX改善**
  - ページネーション機能 (10/15/20/50件表示)
  - ソート機能 (重要度順/新着順)
  - Tailwind v4 アニメーション設定
